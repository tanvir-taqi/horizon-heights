2. How to Install

Option 1: Local Installation

1. Extract the horizon-heights.zip file.

2. Open the extracted folder and locate index.html.

3. Double-click to open it in your default web browser.

Option 2: Hosting on a Web Server

1. Upload the entire extracted folder to your web server using FTP or cPanel.

2. Ensure your server supports HTML, CSS, and JavaScript.

3. Navigate to your domain (e.g., www.yourdomain.com/horizon-heights/).

3. Customization

Editing Styles

To change the design, modify styles/main.css. You can update fonts, colors, and layouts.

All template images are stored in the images/ folder. Replace existing images while keeping the same filenames.

Modifying JavaScript

Scripts for interactivity are inside the scripts/ folder. Edit scripts/main.js to modify functionality.

4. JavaScript Functions

The template includes JavaScript functions that handle interactive elements. Below are explanations of key functions:

Navigation Menu Toggle

This function allows users to toggle the mobile menu:

    function toggleMenu() {
        let menu = document.getElementById('nav-menu');
        menu.classList.toggle('active');
    }

How it works: When the menu button is clicked, it adds or removes the active class, making the menu visible or hidden.

Image Slider

This function controls a basic image slider:

    let currentSlide = 0;
    function nextSlide() {
        let slides = document.querySelectorAll('.slide');
        slides[currentSlide].classList.remove('active');
        currentSlide = (currentSlide + 1) % slides.length;
        slides[currentSlide].classList.add('active');
    }

How it works: The function cycles through slides by removing and adding the active class.

Form Validation

To ensure proper form input, this function checks if fields are empty before submission:

    function validateForm() {
        let name = document.getElementById('name').value;
        let email = document.getElementById('email').value;
        if (name === '' || email === '') {
            alert('Please fill in all fields');
            return false;
        }
        return true;
    }

How it works: If the name or email field is empty, an alert appears, preventing form submission.
